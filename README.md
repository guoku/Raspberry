**Raspberry**
=========

Backend service api/web for guoku good life 100

#系统环境
* Ubuntu 12.04.3 LTS
* MySQL 5.5
* MongoDB
* Nginx 
* Redis
* RebbitMQ

# 后端框架
* django-1.5.8

# 安装方法
```
pip install virtualenv
virtualenv django15
source /path/to/django15/bin/active
pip install -r requirements.txt
```
